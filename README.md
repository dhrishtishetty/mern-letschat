# 💬 Let'sChat

A full-stack **real-time chat application** built using the **MERN Stack**, **Socket.IO**, and **Cloudinary**. Users can chat instantly, share images, edit/delete messages, and securely manage their accounts.

## 🚀 Features

* Real-time messaging with Socket.IO
* User Authentication (Login & Signup)
* Delete User Account
* Edit & Delete Messages
* Image Sharing with Cloudinary
* Emoji Picker
* Online/Offline User Status
* Responsive UI
* Toast Notifications

## 🛠️ Tech Stack

### Frontend

* React
* Vite
* Zustand
* Axios
* React Router DOM
* Socket.IO Client
* React Hot Toast
* Emoji Picker React
* Lucide React

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* Socket.IO
* JWT Authentication
* bcryptjs
* Cloudinary
* Cookie Parser
* CORS

## 📡 REST API Methods

* GET
* POST
* PUT
* PATCH
* DELETE

## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/dhrishtishetty/letschat.git

# Go to the project folder
cd letschat

# Install dependencies
npm install
```

## ▶️ Run the Application

```bash
# Build the frontend
npm run build

# Start the application
npm run dev
```
